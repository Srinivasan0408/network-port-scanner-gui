# Network Port Scanner GUI

## Description
This project is a Python-based network port scanner with a graphical user interface. It scans a target system to identify open ports using TCP connection techniques.

## Features
- Multi-threaded scanning
- Real-time results
- Service detection
- Save results to file

## Usage
1. Enter target IP (127.0.0.1)
2. Set port range
3. Click Start Scan

## Technologies Used
- Python
- Tkinter
- Socket Programming
